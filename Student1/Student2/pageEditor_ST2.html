Rashmi De Silva - Student 2 (Content & Styling Developer)

My Role and Contributions

I was assigned the role of Student 2 in our group project for the module 4COSC011C (Web Design and Development). My responsibilities focused on creating the visual and informational core of the website based on the theme "Life on Land" with a special focus on Saving Endangered Species.

I designed and coded the following pages:

  • Home Page: Includes a full-width autoplay background video and sections for About, Vision, and Mission. Styled using global CSS.
  • Content Page: Rich, scrollable page containing Introduction, Threats, Success Stories, and Get Involved sections, each supported by relevant images.
  • Table Page: Built a structured HTML table displaying data on 10 endangered species, including their common names, scientific names, population estimates, habitats, and conservation statuses.
  • Page Editor: Created this documentation page to explain my work and decisions as Student 2.

Design and Technologies Used

All my pages use consistent styling defined in styles.css, along with shared header, navbar, and footer components. I used HTML5 semantic elements and modern CSS (Flexbox, transitions, gradients) to ensure clarity and responsiveness. Each image includes alt-text, and headings are used appropriately for accessibility.

Key Tags and Concepts

  • <video>, <section>, <main>, <aside>, <ul>, <table>, <footer>
  • Flexbox layout for responsive design
  • Gradient backgrounds, shadows, hover effects, and image borders
  • Scroll behavior for smooth navigation and anchored sidebars

Accessibility Features

  • All images contain descriptive alt attributes
  • Font sizes and contrast meet WCAG accessibility standards
  • Consistent layout with keyboard-friendly links

Validation and Testing

Each page was validated using:

  • W3C HTML Validator for HTML structure
  • W3C CSS Validator for styling rules

I confirmed that all files passed validation with no errors or major warnings. Minor warnings (e.g. experimental properties) were justified by visual enhancements.

Click here to view my Validation Page

Challenges and Lessons Learned

Home Page: Integrating a full-screen background video was difficult across screen sizes. I used object-fit and media queries to maintain scale and clarity.

Content Page: Managing long-form content required careful use of sections and image placement to keep readers engaged and avoid clutter.

Table Page: Making a table mobile-friendly was challenging. I applied responsive widths, scroll overflow, and consistent styling for readability.

These challenges helped me improve in responsive design, semantic HTML usage, and teamwork in collaborative CSS development.

Compliance

All pages were developed in accordance with W3C standards for HTML and CSS. Accessibility was prioritized through semantic markup, alt attributes, and keyboard-friendly navigation. The website adheres to WCAG contrast and heading structure guidelines to ensure usability and inclusivity. Pages were validated through official W3C tools to maintain correct syntax and structure.

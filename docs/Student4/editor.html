Lisara Ransandi - Student 4

My Role and Contributions

As Lisara Ransandi-Student 4 in our 4COSC011C group project on "Life on Land," my main responsibilities were to create the Site Map and the User Profile, and ensure that all other pages were tested and validated properly.

  • Sitemap:A sitemap visually represents the structure and navigation paths of a website, helping users understand how different pages are connected.
  • User Profile: The user profile page displays personal details and preferences specific to each user.
  • Testing: Checked responsiveness, accessibility, and ensured internal links and visuals functioned across all browsers.
  • Assisted: Cross-tested group members’ work to confirm layout consistency and design integrity.

Testing and Validation

I used the following tools and steps to verify code quality and user experience:

  • W3C HTML Validator
  • W3C CSS Validator
  • Tested responsiveness with browser tools (Chrome DevTools, Firefox Inspector)
  • Checked all hyperlinks, forms, buttons, and multimedia for expected behavior

Click here to view my Validation Page

What I Learned

This project taught me how to properly validate HTML and CSS, debug layout issues, and understand the importance of user experience. I also improved my ability to document and report errors while ensuring accessibility standards were met across the site.
